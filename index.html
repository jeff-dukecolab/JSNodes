Nodes

What are they?

pieces of the DOM

The DOM is the Document Object Model, which you can think of as a tree.

The website (document) is made up of small pieces, called nodes. Everything that has a tag is a node. This paragraph is a node.

We can create, modify and remove nodes with javascript without reloading the page!
